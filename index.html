About / notes
  • Issue: paste an npub1... (or 64-char hex) and press Issue to generate that profile's card. Processing is identical whether the key comes from NIP-07 or manual entry — no private key required.
  • Get npub from NIP-07: pulls your public key from a NIP-07 extension (nos2x, Alby, …) into the input box. It does not sign anything.
  • NIP-05 is actually verified against /.well-known/nostr.json (green ✓ = verified, red ✗ = mismatch, no mark = could not check).
  • Share to Nostr (appears when a NIP-07 extension is detected): uploads the card image (NIP-96) to a host you choose, then publishes a hashtagged kind:1 note to the relays above. Every signature is approved in your extension; no private key leaves it.
  • This is a fan card for fun, not an official ID.
  • If an avatar host doesn't allow CORS, the avatar may fall back to a placeholder in the downloaded image.
